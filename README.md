# MasomoBora

A full stack e-learning platform.

## Core Functionality:

- Auth:

  - Register
  - Login
  - Refresh access tokens
  - Logout
  - Profile

- Course:

  - Create course
  - Get course categories
  - Get all courses
  - Get courses by category
  - Get course details
  - Update course

- Chapters:
  - Create chapter
  - Get all chapters
  - Get chapter details
  - Update chapter details
  - Toggle chapter published status
